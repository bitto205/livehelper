# LiveHelper

抖音直播间消息采集与直播用工具

## 功能概览

- 实时接收直播消息：弹幕、礼物、点赞、进场、关注、在线人数等
- 多线路监听，按场景切换：
  - 线路 1：页面 JS Hook
  - 线路 2：Playwright + WSS 帧解析
  - 线路 3：mitmproxy local 进程拦截
  - 线路 4：直播伴侣 patch + `proxy_shell` IPC
- 统一 protobuf 解析入口：`listener/LiveProtobuf.py`

## 工具介绍

### 直播备忘录

直播备忘录是一个可以单独打开的小窗口，用来把直播中值得留意的互动自动整理成清单。

开播时如果观众送礼、关注、点赞，或者加入粉丝团，备忘录会自动新增一条记录。你可以把它当成直播时放在旁边的“待处理列表”：看到一条、处理一条，点一下记录或右侧的 `×` 就能把它划掉。

它适合用来提醒自己：

- 谁送了礼物，等下要不要感谢
- 谁关注了直播间，是否要互动一下
- 谁加入了粉丝团，是否要特别欢迎
- 哪些临时事项需要自己手动记一下

备忘录也支持一些常用设置：

- 礼物可以按“同一个人送的同一种礼物”自动叠加数量
- 点赞可以按用户自动叠加数量，避免刷出一长串重复记录
- 礼物可以设置最低钻石数，低于门槛的礼物不进入备忘录
- 可以手动添加自定义备忘，比如“5 分钟后发福袋”“提醒讲下一款”等

### 弹幕机

弹幕机可以把直播间里的弹幕、礼物、点赞等消息显示成独立窗口里的可见弹幕。打开工具并开始采集后，就可以把它作为直播辅助显示窗口使用。
