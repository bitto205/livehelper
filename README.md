# LiveAll-in-one/LiveAIO

LiveAIO 是一个给抖音直播使用的辅助工具。它可以连接直播间，接收弹幕、礼物、点赞、关注、进场、在线人数等消息，再把这些消息交给不同的小工具使用。


## 主要能做什么

- 连接抖音直播间，实时接收观众互动消息
- 支持网页直播间，也支持直播伴侣相关的监听方式
- 自动识别弹幕、礼物、点赞、关注、粉丝团等常见事件
- 提供直播备忘录，把重要互动变成可处理清单
- 提供弹幕机，把直播消息显示成透明悬浮弹幕窗
- 支持颜色主题和关闭到任务栏

## 第一次使用

1. 启动软件后，先进入“主页”。
2. 选择一条适合自己的监听线路。
3. 如果使用线路一或线路二，需要先登录一个抖音账号。
4. 填写直播间 ID，保存后点击连接。
5. 连接成功后，再到“工具”页面打开需要的辅助工具。

建议使用小号或非直播号登录。连接过程中避免用这个登录账号进入其他任何直播间，避免影响消息采集。

## 线路怎么选

### 线路一

适合网页直播间监听。需要登录抖音账号，连接方式比较直接。

### 线路二

适合网页直播间监听，也是日常优先推荐的线路。需要登录抖音账号，稳定性通常更好。

### 线路三

监听直播伴侣，不需要登录抖音账号。使用时不能同时运行代理软件。连接后不要关闭直播伴侣，也不要中途关播，否则需要重新开播再连接。

### 线路四

监听直播伴侣，不需要登录抖音账号，但需要先对直播伴侣执行 patch。这个线路依赖伴侣侧的辅助程序，当前不行可以先使用线路一或线路二。

## 工具介绍

### 直播备忘录

直播备忘录是一个可以单独打开的小窗口，用来把直播中值得留意的互动自动整理成清单。

开播时，如果观众送礼、关注、点赞，或者加入粉丝团，备忘录会自动新增一条记录。你可以把它放在旁边，当成直播时的“待处理列表”：看到一条、处理一条，点一下记录或右侧的 `×` 就能把它清掉。

它适合用来提醒自己：

- 谁送了礼物，等下要不要感谢
- 谁关注了直播间，是否要互动一下
- 谁加入了粉丝团，是否要特别欢迎
- 哪些临时事项需要自己手动记一下

备忘录还支持这些设置：

- 礼物记录可以开启或关闭
- 同一个人送的同一种礼物可以自动叠加数量
- 可以设置最低钻石数，过滤掉低价值礼物
- 关注记录可以开启或关闭
- 点赞记录可以开启或关闭
- 点赞可以按用户自动叠加数量
- 可以手动添加自定义备忘，比如“5 分钟后发福袋”“提醒讲下一款”

### 弹幕机

弹幕机可以打开一个透明悬浮窗口，把直播间消息显示成可见弹幕。

弹幕机支持显示：

- 普通弹幕
- 礼物消息
- 关注消息
- 点赞消息

你可以分别控制每一类消息是否显示。礼物可以设置最低钻石数，点赞可以设置数量阈值，也可以开启累加模式：同一个用户的点赞数量累计到阈值后再显示，避免点赞消息刷屏。

弹幕机还支持给不同类型的消息加短后缀，比如在礼物后面加一句“感谢支持”，在关注后面加一句“欢迎加入”。

## 设置

设置页里可以切换软件颜色主题，也可以设置关闭窗口时是否缩小到任务栏。

如果开启“关闭时缩小到任务栏”，点关闭按钮后软件不会直接退出，而是缩到系统托盘中继续运行。你可以从托盘菜单重新打开窗口，也可以控制部分工具。

## 使用建议

- 初次使用建议先试线路二。
- 如果只是想做网页直播间辅助，建议不用直播伴侣线路。
- 如果没有能登录的账号或者有其他的账号需要，再考虑线路三或线路四。
- 礼物相关统计以抖音为准！软件难免会出错。
